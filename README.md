# Contact Hub

## Overview
Contact Hub is a project designed to manage and organize your contacts seamlessly. It allows users to add, update, search, and delete contacts with ease. The application provides a user-friendly interface and efficient data management capabilities.

## Features
- **Add Contacts**: Easily input and save contact information.
- **Update Contacts**: Modify existing contact details whenever necessary.
- **Search Functionality**: Quickly find contacts using search filters.
- **Delete Contacts**: Remove contacts you no longer need.

## Technologies Used
- **Front-end**: HTML, CSS, JavaScript
- **Back-end**: Node.js, Express
- **Database**: MongoDB

## Getting Started
1. Clone the repo: `git clone https://github.com/pedrobirindelli/contact-hub.git`
2. Navigate to the project directory: `cd contact-hub`
3. Install dependencies: `npm install`
4. Start the server: `npm start`
